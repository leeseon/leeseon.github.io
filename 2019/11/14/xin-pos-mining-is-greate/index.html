为什么说Mixin节点是一门长期靠谱的项目

发表于 2019-11-14 更新于 2023-02-23

缘起

昨天问起同事，我前天写的文章牛不牛？本来以为会收到溢美之词，结果他直接来了一句TLDR（太长没有看完）。错愕之余，摒掉“明珠暗投”之感后，我认真反省自己：对啊，好好的一件事，为什么要用这么繁复的表达？细思极恐：看来还是吃了财商教育不足的亏，从小学到的都是“肉食者鄙”、“小人喻于利”之类的观点，结果自己发现了一个感觉还不错的靠谱项目，想拉人入伙、找人投资却抹不开脸面直接谈钱，居然玩煽情。这年头，读你一篇长文是给你面子，你还这么绕，太浪费大家的时间了。对嘛，我又不骗人，为什么要不好意思谈钱嘛。

好吧，这一次我坦坦荡荡、堂堂正正的来谈谈“赚钱”的事。

电梯测试

我试着简单点，看能不能尽量在30秒内引起大家的兴趣，也就是传说中勾引VC风投的“电梯测试”。

如果只让我说一句：
我有一个基于区块链的去中心化支付平台公司的早期入股机会，投入大概要两千万，回报率大概有20倍（风投基金的锁定期常规就是5+2年）。

如果这时你没有打断，我会补上两句：

  • 其实简单点说，我发现了一个百倍币，当然保守估计也有10至15倍增长空间
  • 而且如果有一万个币，就可以抵押币成为记账节点，七年能有73%左右的额外的币的收益
    两者相乘保守也能有20倍收益

如果依然还有兴趣的话，我会这样解释：

  • 为什么说是百倍币呢？因为它完美地解决了一个真正的痛点：对于数字货币钱包来说，安全与方便很难兼顾，但现在能做到，仅靠一个手机号码再加6位数字的PIN码，简单安全的存储几乎所有数字货币。
  • 此外，你只需要知道对方的手机号码，就能准确无误地给对方实时转账任何数字货币
  • 并且，从技术设计上，几乎不会发生丢币或者跑路的情况

如果还没有能说服你，我会这样展开：

  • 此外因为钱包背后本身就是一个数字货币储值平台，天然能提供转账与支付功能，由此可以很正常的扩展出交易所、借贷之类的金融生态
  • 并且钱包本身也是一个聊天软件，除了社交之外更能天然的扩展出知识付费，邀请返现之类电商玩法
  • 更牛的是，以上所有功能并不是停留在白皮书阶段，而是已经做出来了开箱即用，并且任何人都可以像为微信、支付宝开发小程序一样，在上面扩展自己的功能，有很大的商业想象空间

而且，我还能从估值上定位：

  • 现在这个项目估值不过2.5亿美金，流通市值不到1.3亿，与支付宝和微信千亿级美金的估值相差太远，我们就不用去幻想这些了
  • 如果与仅仅做数字货币支付业务Circle公司80亿美金的估值对标，也是严重低估的

横向比较：

  • 竞品都不太好，闪电网络技术门槛高，Cosmos还基本没有人用，Polkadot还没有发布正式版
  • 此外竞品使用的BlockChain技术，而这个项目使用的是DAG，BlockChain需要等待上一个区块生成才能生成下一个区块，而DAG像一个网，可以在任何点上向外扩展，不用等待上一个区块生成，更适合支付这种对实时要求性高的项目

生态位：

  • 头部产品会引起更多用户聚集的马太效应
  • 飞轮效应：虽然现在Mixin的用户还不太多，但是正在有更多的推广将用户吸引过来

blabla、、、

但是

然而VC是不会将我请进他们的办公室详谈、签合同与开支票的。因为还有一个问题的答案必定让他们勃然大怒:

  • 请问，你需要多少钱，准备怎么开展这个项目？

“呃，我只要两千万RMB，人并不需要其他人，我一个人、、、、”

“滚！”

诚如你所料，其实上面只是一个玩笑，我又没忍住开始插科打浑了。因为这本来就不是一个找风投的项目，只是借用了风投的“电梯测试”来展开场景。毕竟我现在在做的事，其实与风投基金比较象，也是找钱，投在项目里，然后等时间给出结果。

不过更应该说，这个更像是一次私募，只是不像一般的股票基金型的私募，因为POS机制的存在，有点象是去买股票，卖方说如果你买一万股并承诺七年锁仓的话，按六折（1/1.7~=0.6）卖你。

神转折

私募经理募资时只会向你展示战绩与选股理念，但不会告诉你投资标的与配比。我虽然有投资业绩但不能拿来一说，因为在区块链的世界里，那有点小儿科了:(

不过我直接告诉你，我为什么这么选也是有坦白的动机的：

  • 我还准备在区块链的投资世界里活很久，所以一开始亮明观点与看法，如果万一成了这将是一个历史性的背书，也将是一次信用的积累
  • 而且如果你真的认同了我的理念但没有投我，而是自己去买了XIN，甚至自己去挖XIN，对我也没有什么坏处，因为我现在本身也是XIN的长期持有者，自然也能从中受益
  • 除了能运营XIN节点外，我对资方的价值主要是能帮你锁住钱，不会让你因为暴涨而卖了，失去了本来应该获得的巨额利润，是那个财富的看门人

补充

这是上一篇文章思路的简化版，所有有很多前提是没有论证并直接认为是成立的：

  • BTC不会死，而会越来越好
  • BTC和其它数字货币的支付业务是一个必然刚需
    所有请谨慎判断

风险

还是需要再郑重警示一下：区块链投资现在依然存在高风险，请使用闲钱投资。此外你最好是一个合格的投资人。

我吹牛其实蛮靠谱的，说不定它真的实现了呢
