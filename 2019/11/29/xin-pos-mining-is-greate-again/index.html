再说说Mixin节点的事

发表于 2019-11-29 更新于 2023-02-23

换个角度

如果我是 GP ，应该怎样向 LP 们解释为什么要做 Mixin 节点？

Mixin 是什么？

官宣如下：

Mixin 是一个免费、快速的点对点跨链数字资产交易网络，可帮助其他区块链分布式账本获得超高 TPS、亚秒级确认、零手续费、加强隐私、无限扩展的能力。

可以不精确的理解为：能完成比特币及其它一切数字资产的支付平台。衍生一点，就是它同时提供了一个去中心化的钱包功能。

Mixin 节点是什么？

因为 Mixin Network 是一个去中心化的数字资产交易网络，所以它不能由一个统一的中心化服务来进行网络的记账服务，它需要由一系列独立的服务器为这个网络每一笔转账提供大多数人同时认可的信任背书。简单来说，为整个 Mixin 网络的转账提供记账服务的服务器提供方，就视作 Mixin 的一个节点；而一笔账能被大多数人同时认可的规则就被称为网络的共识机制。

XIN 与 POS 挖矿

Mixin 使用的是 PoS + Asynchronous BFT 的共识机制，撇开太专业的 Asynchronous BFT ，这套共识机制的核心是POS。相较于 BitCoin 所使用 POW（工作量证明机制）：使用算力无差别的争夺记账权，POS（权益证明机制）无需通过算力来争夺记账权，只取决于拥有权益的多少。为了量化权益及记账产生的激励（收益），Mixin 发行了它的代币，也就是XIN。

简单来说，需要抵押一定量的 XIN 成为节点，就可以参与 Mixin 的 POS 挖矿，挖矿的收益也是 XIN 。

  • 代币总量：恒定 100 万
  • 加入节点无需许可，可匿名参与，抵押上一年流通量的 2% 即可加入。例如第一年加入全节点需要 10,000 XIN，第二年可能需要 11,000 XIN 左右
  • 退出节点无需许可，退出节点需要扣除抵押的 1% 作为手续费

收益估计

全节点挖矿奖励

  • 初始矿池 500,000 XIN，每年从剩余矿池挖 10% 分给节点，其中全节点占 90% 轻节点占 10%，理论上每年挖矿的数量会逐年减少但永远也挖不完。

全节点收入：第一年45,000个，第二年40,500个每年递减。截至目前，共有31个 Mixin 节点，如果现在做节点（不考虑有新加入与退出）按32个节点算：

以上的收益是根据币本位计算的，如果长期来看，币价上涨了 10 倍则总收入会是 1.7x10=17 倍，币价上涨 100 倍则总收入会是 1.7x100=170 倍、、、

XIN 的估值模型与空间

XIN的估值模型：

  • Mixin 的用户数
  • Mixin 上的数字货币储值规模

根据梅特卡夫法则，一个网络的价值与用户数的平方成正比。用户数增大，用户之间的连接可能性是以用户数的平方规模增大的，而用户连接的可能性是网络价值的核心体现。

此外一个支付平台的价值也与网络中沉淀的资金总量相关，有千亿资金的支付宝与万亿资金的支付宝估值肯定是不同的。资金规模也与用户数正相关，同时用户数多了会间接增加ARUP（人均储值）。

简单粗估一下：XIN 的估值增长应该大于用户数的平方而趋近于用户数的立方，即用户数增长 10 倍，XIN的估值大于 100 倍而趋近于 1000 倍。这仅是理想模型，不过趋势与上两个因素紧密相关。

从对标上估算：

  • XIN全量估值不过2.2亿美金，流通市值不到1.1亿，与支付宝和微信千亿级美金的估值相差太远
  • 如果与仅仅做数字货币支付业务Circle公司80亿美金的估值对标，有 30~40 倍的想象空间
  • 此外对比 BTC 市值，XIN 的估值基数更小，增长空间更大

Mixin 的商业逻辑

Mixin 本身是一个对用户基本不收费，形成生态后，对生态中的商家收费的商业模式（可参照所有支付平台）：

  • Mixin Network 收取额外的矿工费，这个是 to C 的，但现阶段有相应的返现，暂时算免费
  • Mixin Messenger 的钱包功能，是一个发展 C 端用户的关键点：刚需、高频、同时能兼顾安全与方便数字货币钱包
  • 在钱包功能外还直接附送了一个更高频的社交功能（聊天），并直接借此打造出了最自然的转账应用场景：与联系人在聊天中转账，知道对方手机号就能转账
  • 此外提供良好的 API ，供大家生成类似于微信小程序的机器人服务，进一步扩展生态入口

Mixin 的竞争优势

Mixin 的优势主要集中在以下两个方面：
数字资产保存一直是一个痛点：放交易所会面临被盗与跑路的风险，自己保存技术门槛太高，助记词与私钥并不是适合非专业人士，而且就算专业人士成本也会太高。

  • Mixin 使用手机号与 6 位数字 PIN 码的方式，解决了数字资产安全与方便地保存问题，这是现阶段最优的解决方案没有之一

数字资产（尤其是BTC）的小额、快速支付是另一个天然的需求，随着大家对数字资产的接受程度加深，跨链 的支付网络已经是一个竞争激烈的战场，不过现在看来Mixin是最优且落地的解决方案没有之一。

投资思路

从逻辑上确认 XIN 是一个值得长期投资的标的，对于短期币价上的风险无从提前判断，只有长期持有才可能收获巨大收益。
那既然长期持有何不抵押一万个 XIN ，参与节点收获额外的收益，并对冲一定的币价波动

Mixin 挖矿目标用户

非目标用户

  • 分不清楚区块链与数字资产之间关系者，多半是被近期区块链新闻吸引的关注者
  • 对数字资产持怀疑论者
  • 对数字资产支付功能持怀疑论者
  • BTC “一神论者”，不相信比特币之外任何数字资产者
  • 关注短期币价者
  • 短期炒币者，退出手续费为1%，即 100 个 XIN 太贵

目标用户

  • 认同区块链是一个长期赛道
  • 认同 Mixin 是符合区块链技术发展方向
  • 认同 Mixin 商业模式与竞争优势
  • 能长期持有者

风险评估

最大的风险，可能是 Mixin 的用户发展得不好，没有人使用的平台没有价值。参考 Mixin 的竞争优势，我认为安全毕竟最重要的刚需，所以随着用户的理解加深，自然会发现 Mixin 的确是最好的钱包并使用， 这个也只是我主观​的判断，所以风险客观存在。

第二个风险有否有可能出现竞品而将 Mixin 的用户争夺走，我的主观判断是不大，除非它能做得比 Mixin 好很多，从而产生的收益大于迁移成本，但不除非技术发展会有这种可能性的存在，只不过可能性很小。

FAQ

感觉 Mixin 挖矿主要收益还是来源于币价的增长，币本位的收入不太多，那买 XIN 就好，为什么还要挖矿？

  • 七年复合收益率大于 8%，对币本位来说其实已经很高，如果有 BTC 有这个收益率会很惊人
  • 挖矿不适合小资金量及短期投入者，这部分用户可能众筹节点，或者买 XIN 就好
  • 如果是大资金并且长期投资，额外的收益是相当可观的

帮助区分一下 Mixin Network、Mixin Messenger 及 ExinOne 与 FoxOne 等？
新用户一下接受不了太多，那先暂时不精确的类比一下吧：

  • Mixin Network 是后台服务，相当于支付宝背后的一堆服务器，用于记录各种转帐。
  • Mixin Messenger 是一个移动端的 App，相当于手机上的支付宝应用，同时还提供端到端的加密聊天功能，最大的亮点是包含有一个去中心化的多数字资产的钱包。其实反着想想就能很好理解了，支付宝最主要的功能也就是一个钱包，只是管理的是人民币而已。
  • ExinOne 是一个基于 Mixin Messenger 上的机器人，提供数字货币的交易所功能；相当于支付宝上的小程序，是 Mixin 生态的一部分。
  • FoxOne 是另一个移动 App ，也是使用 Mixin Network 的平台功能的去中心化交易所，由另一个团队开发，有额外的交易功能。
