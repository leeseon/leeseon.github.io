Mixin Messenger 伪装成聊天工具的区块链定投神器（下）

发表于 2019-09-03 更新于 2023-02-23

闲扯

昨天又翻了一下李笑来在2018年11月份写的《这此年我看到的所谓区块链》，这也算是针对 Mixin Network 的正式“软文”了。当时 XIN 正好是价格的低点啊！我居然没有真的去找一下 Mixin Network 的白皮书去看看，看来思想不 Opening，而且认知速度慢还是很耽误赚钱的。

不过世界本来也不存在所谓的“错过”，自以为是错过了，那肯定是一厢情愿而已，因为机会一直在那里，你没有捉住，只能说当时你配不上它。就如现在3美金的EOS就在那里，或者说1万美金的BTC也就在那里，等过了几年，当它们价格飞上天时，你好意思说你错过了吗？没有，你只是当时（就是现在）你不认为它们会飞上天而已，事实证明你配不上它。所以你本不会错过一些什么。

好在我还算运气不错，现在发现了 XIN 的价值也不晚啊，在区块链的世界，机会多的是，真正的大机会也并不少，随时都来得及。

支付宝+微信

现在仔细想想，感觉 Mixin Network 还真的做成了币圈支付宝啊。让我们试着回忆一下支付宝早期的事吧，当时银行还没有什么电子银行，更没有什么手机银行，银行间转帐基本上要去柜台，而且还很慢（一两天吧）。后来有了支付宝，虽然一开始将钱转到支付宝平台内还比较麻烦，但一旦钱进入到支付宝中，那在支付宝的各个帐号中转来转去就相当的简单与快速，而且转帐费也是免费的。那进一步想想，虽然支付宝平时多半是用于单币种，但是一个全球性的多币种的支付宝，也是理论上有可能的，对吧！那顺着这个思路，就会发现 Mixin Network 是就一个真正的全球性、多币种的数字资产的“支付宝”平台啊。

你可以将 BTC、EOS、ETH 之类的各种链想象成一个个独立的“银行”，它们发行各种数字资产（为了免于歧义，还是不要称之为货币），因为这些所谓的“银行”自身的运行逻辑或安全机制的限制，用户在各种银行间转帐的时间与费用是不同的：例如 BTC “行”的转帐时间是一个小时，几十块钱， EOS与ETH “行”转帐需要几分钟，EOS免费而ETH大要几毛钱的手续费。不过一旦当你将你在 BTC、EOS 等“行”的钱转到了 Mixin Network 上，那所有 Mixin Network 的用户间的转帐就直接变成实时的，而且还是免费的了。你说这不是一个币圈支付宝是什么啊。

不过，这个币圈支付宝也不能解决，当需要将 Mixin Network 上的资产提回各种“银行”所需要的时间与费用的。例如，将BTC 提出，依然是需要一个小时的时间与几十块的费用的。仔细想想支付宝的大额提现与提现手续费你就懂了吧。

当然，你此时一定会冒出“那，Mixin 不就是一个中心化的节点（平台）了吗？”这样的问题。嗯，你是对的，的确是这样的，只不过 Mixin Network 是一个按去中心化方式运营的中心型节点。至于它本身是怎样去中心化运营的，这个我以后会慢慢讲，有兴趣大家也可以先去读读 Mixin 的白皮书。

此外，很多人有点分不太清楚 Mixin Messenger 与 Mixin Network 的关系。其实 Mixin Network 是一个后端的保存资产与帐户关系的平台，而 Mixin Messenger 则是使用 Mixin Network 接口的官方客户端App。这就如同手机上的支付宝应用 APP 与阿里无数台服务器构成的支付宝平台之间的关系一样。不过比较另类的是 Mixin Messenger 没有作成手机支付宝的样子，而是做成一个聊天工具。哈，这是要支付宝 + 微信合体吗？

Mixin Network 的创始人冯晓东有一次聊到，他并不喜欢别人将 Mixin Messenger 说成是币圈微信，感觉这样不能显示 Mixin 的牛叉。拜托，要不要这么傲骄，你怎么不上天？那我将 Mixin 还原为币圈支付宝+微信总行了吧。

交易所+余额宝+定投

就算稍有一点商业上的嗅觉，也会发现有趣的问题就来了：一旦有了币圈的支付宝，那基于这个支付平台能做什么来赚钱呢？如果你只想到了做电商，哈，看来商业嗅觉不及格啊。离钱这么近，当然是做金融啊！嗯，没错，果然聪明的人都想到一块了：

Mixin 上最好用的交易所、余额宝及定投神器三合一的服务 ExinOne 就已经在那儿等你了，你必须拥有。在 Mixin 中所有的服务都被称作机器人，所以你现在只需要在 Mixin Messenger 中搜索 ExinOne（ID：7000101276） 将其添加为联系人即可。

以后只要点下 Exin 那个小机器人图标，你就会看到：

C2C(法币交易)、币币交易、定投设置就都在这里了，功能都很易用，随便摸索一下就能在两分钟之内花 50RMB 买到你人生的第一笔 BTC 了。

此外 Exin 还提供了币余宝的功能，平时不用的币放在这里居然能币生币。

Exin 的功能都很简单，应该可以直接上手，如果对一些具体的细节不了解，可能通过金马的文章https://bihu.com/article/1539026717了解一下。
