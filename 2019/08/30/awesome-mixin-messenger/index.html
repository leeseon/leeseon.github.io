Mixin Messenger 伪装成聊天工具的区块链定投神器

发表于 2019-08-30 更新于 2023-02-23

Mixin Messenger 是我一不小心差点错过的一款神器：居然是一个伪装成聊天工具的区块链钱包+交易所+定投工具+理财工具+社区工具的神器！

做为币圈老鸟，早在2017年底我就知道了 Mixin Network 要ICO的消息，不过那时我正在为BTC的价格HIGH的不行，而且除了EOS基本上不参加任何ICO，毕竟有太多的ICO不靠谱，再加上当时20EOS/XIN的价格有点让人感觉太高（哈，我还是太没有想象力了）。

再加上Mixin的技术是使用了另类的 TEE-BFT-DAG 让我一个 BlockChain 的原教旨主义者有点举棋不定，于是也就错过了，甚至在漫漫熊市都没有想起来看它一眼。

哪怕它早在今年年初主网上线了也基本上在业内动静不算太大，毕竟那时正熊得大家灰头土脸的吧。

不过也就是今年的7月3日之后，我终于发现了它的牛逼之处，原来居然是一个最好用与安全的多币种钱包，哈，你居然伪装得这么好，真是服了。

钱包

我在上一篇文章《来，这一次让我们慢慢变富》中说过，长期严肃地数字资产投资最先需要搞定钱包与交易所。但是个人管理钱包其实是有很大的坑的：从比特币创建伊始，私钥的安全保存就是一个极大的难题，有很多人就是因为不小心丢掉了私钥，结果币在那里却取不出来。

其实这主要是因为比特币的设计中有一个不符合常识的地方：

  • 现实世界里每个人是先拥有了一个银行卡号，再设置一个取款密码，然后使用这种方式来管理资产的，如果不慎密码忘了，还可以通过身份证明去银行找回密码。
  • 但是比特币的世界里却不一样：每一个地址（相当于银行卡号）都是根据私钥（相当于取款密码）算出来的，掌握了私钥就掌握了一切。现实世界中你只是丢了取款密码别人不知道你的帐号，还没有办法取走你的钱，但是区块链世界中，如果私钥忘了，是没有一个如同银行一样的中心机构能帮你找回密码的，也就是说没有任何人（对！就是任何人，也包括你自己）能动用那个帐号中的钱；如果是私钥被偷了，也就意味着帐号里的钱也会一起被偷。
  • 更可怕的是：如果你的币被人偷走了，极大可能要不回来，因为比特币的世界里交易是不可逆的，而且地址本身是匿名的。

嗯，我知道你读到这里大概会想问，既然私钥是唯一获取数字资产的凭证，那我怎样保证我的私钥不被人猜中；或者如果我如果猜中了别人的私钥，岂不是直接就能拿走别人的币？
Good Question! 对！说中了要害了！所以为了安全，私钥使用了一个巨大的可能空间，有多大呢？ 2的256次方，相当于10的77次方，也就是1的后面要写77个0才可能表式出来。
嗯，我知道这样说还是不太直观，那再说两个数字吧，地球的原子总数是10的50次方，而可观测宇宙的原子数是10的80次方。
如果地球上的每个原子都是一台计算机，并且每个原子每秒可以算1亿次私钥，那也需要3671亿年才能穷举完所有的私钥，而现在已知的宇宙也只有137亿年而已，那时太阳系是否存在都不一定了。

因此比特币的私钥几乎是不可以被人猜中的，但是这样也带来了一个很大问题，一个如此大的数如何保存与记忆呢？这个当然难不倒程序员啊，所以大家也就会见到 KyEvj7S2ZLmbdxvc1f8YXmtnEp4kgTvwBo2Ck75qtr83Yn52FJRK 与 5JGqsmxWokSxsyAsyno7h3vRJBDBMGcnMa5ocbTUPd3xQaku8NJ 的东东了。

更要命的是，为了安全，一个地址可以多次收款，但转一次帐之后最好就不要再使用了。所以每个人其实是需要很多私钥的，而管理一系列私钥的软件就被称之为钱包。为了让钱包的管理方便也就有人发明了助记词之类的东西，但这依然从本质上解决不了管理起来太麻烦的问题。（关于钱包的事我以后再找个机会仔细讲，今天先不展开了）

方便与完全好象的确就是不可兼得的鱼与熊掌，不是吗？

不过，你也许会问，难道真的不可兼得吗？难道就不能像银行一样，它来管理帐号，我只负责记密码就行了吗？ Great Question! 你又一次命中了要害，恭喜你！不过不是不想啊，是有一定难度：

  • 得有一个像银行一样的中心化的节点来保存你的资产，例如现在的交易所就是替你保管了你的资产，但是交易所很难自证清白不跑路啊。
  • 其实这两年比较热所谓的跨链项目，他们的目标也是来做这个事的：怎么在自证清白的情况下保存大家的资产，只是进展太缓慢了，现在也没有办法用。

然而好在聪明人是很多滴，比如 Mixin Network 的创始人冯晓东：他直接跳出了 BlockChain 的思维窠臼，直接用比较另类的 TEE-BFT-DAG 方案就将上面需要所谓跨链项目来解决的问题给破解了：

  • TEE-BFT-DAG 能保证每笔交易（转帐）不能做假，而且可回溯
  • 基于 POS 的抵押机制与多重签名从博弈论上保证了节点做恶的代价太高而不划算

上面的技术细节太多，以后我再详细讲吧。

So，因为上面这个关键问题的解决，每个人就真的能放心的将自己的数字资产存储在 Mixin Network 中了，而且钱包的问题也一并解决了：

  • 私钥让 Mixin 去管，我们只需要记住自己的密码就好，对，就是那个PIN，6位数字，在人类大脑可以记住的范围内
  • 那帐号呢？帐号就是你的手机号，先这么理解吧（其实是为每一个手机号分配了一个很长很长的ID）
  • 手机号丢了是可以补办的，帐号不会丢
  • 你只要手机号不被劫持，帐号是别人动不了的
  • 手机号被劫持了，也能通过“紧急联系人”找回帐号
  • PIN很关键，不能忘记不能丢，丢了任何人都找不回资产了，这个与丢了私钥一样

重要的事说三遍：

  1. PIN不能忘记不能丢
  2. PIN不能忘记不能丢
  3. PIN不能忘记不能丢

还有个重要的事也说三遍吧：

  1. 记得设置一个“紧急联系人”
  2. 记得设置一个“紧急联系人”
  3. 记得设置一个“紧急联系人”

安装

说了这么多，那还是赶紧安装一个 Mixin Messenger （Mixin Network 的官方入口）App吧。

Android 用户比较简单，直接在应用宝上搜索就好。

iOS 的用户麻烦一点点因为app store上有两个版本，如果你有app store的海外（非中国）帐号，那就能下载到功能完全的版本，它自带钱包，图标是白底蓝色 logo 。

如果你只有中国帐号，那就只能下载到不带钱包的畅聊版，图标是蓝底白色 logo 。想使用钱包功能需要额外关注一个小机器人。

紧急联系人在头像=>安全与隐私=>紧急联系人 里设制。
