再说说Mixin节点的事

发表于 2019-11-29 更新于 2020-12-16

换个角度

如果我是 GP ，应该怎样向 LP 们解释为什么要做 Mixin 节点？

Mixin 是什么？

官宣如下：

Mixin 是一个免费、快速的点对点跨链数字资产交易网络，可帮助其他区块链分布式账本获得超高 TPS、亚秒级确认、零手续费、加强隐私、无限扩展的能力。

可以不精确的理解为：能完成比特币及其它一切数字资产的支付平台。衍生一点，就是它同时提供了一个去中心化的钱包功能。

Mixin 节点是什么？

因为 Mixin Network 是一个去中心化的数字资产交易网络，所以它不能由一个统一的中心化服务来进行网络的记账服务，它需要由一系列独立的服务器为这个网络每一笔转账提供大多数人同时认可的信任背书。简单来说，为整个 Mixin 网络的转账提供记账服务的服务器提供方，就视作 Mixin 的一个节点；而一笔账能被大多数人同时认可的规则就被称为网络的共识机制。

XIN 与 POS 挖矿

Mixin 使用的是 PoS + Asynchronous BFT 的共识机制，撇开太专业的 Asynchronous BFT ，这套共识机制的核心是POS。相较于 BitCoin 所使用 POW（工作量证明机制）：使用算力无差别的争夺记账权，POS（权益证明机制）无需通过算力来争夺记账权，只取决于拥有权益的多少。为了量化权益及记账产生的激励（收益），Mixin 发行了它的代币，也就是XIN。

昨天又翻了一下李笑来在2018年11月份写的《这此年我看到的所谓区块链》，这也算是针对 Mixin Network 的正式“软文”了。当时 XIN 正好是价格的低点啊！我居然没有真的去找一下 Mixin Network 的白皮书去看看，看来思想不 Opening，而且认知速度慢还是很耽误赚钱的。

不过世界本来也不存在所谓的“错过”，自以为是错过了，那肯定是一厢情愿而已，因为机会一直在那里，你没有捉住，只能说当时你配不上它。就如现在3美金的EOS就在那里，或者说1万美金的BTC也就在那里，等过了几年，当它们价格飞上天时，你好意思说你错过了吗？没有，你只是当时（就是现在）你不认为它们会飞上天而已，事实证明你配不上它。所以你本不会错过一些什么。

好在我还算运气不错，现在发现了 XIN 的价值也不晚啊，在区块链的世界，机会多的是，真正的大机会也并不少，随时都来得及。

支付宝+微信

现在仔细想想，感觉 Mixin Network 还真的做成了币圈支付宝啊。让我们试着回忆一下支付宝早期的事吧，当时银行还没有什么电子银行，更没有什么手机银行，银行间转帐基本上要去柜台，而且还很慢（一两天吧）。后来有了支付宝，虽然一开始将钱转到支付宝平台内还比较麻烦，但一旦钱进入到支付宝中，那在支付宝的各个帐号中转来转去就相当的简单与快速，而且转帐费也是免费的。那进一步想想，虽然支付宝平时多半是用于单币种，但是一个全球性的多币种的支付宝，也是理论上有可能的，对吧！那顺着这个思路，就会发现 Mixin Network 是就一个真正的全球性、多币种的数字资产的“支付宝”平台啊。

你可以将 BTC、EOS、ETH 之类的各种链想象成一个个独立的“银行”，它们发行各种数字资产（为了免于歧义，还是不要称之为货币），因为这些所谓的“银行”自身的运行逻辑或安全机制的限制，用户在各种银行间转帐的时间与费用是不同的：例如 BTC “行”的转帐时间是一个小时，几十块钱， EOS与ETH “行”转帐需要几分钟，EOS免费而ETH大要几毛钱的手续费。不过一旦当你将你在 BTC、EOS 等“行”的钱转到了 Mixin Network 上，那所有 Mixin Network 的用户间的转帐就直接变成实时的，而且还是免费的了。你说这不是一个币圈支付宝是什么啊。

不过，这个币圈支付宝也不能解决，当需要将 Mixin Network 上的资产提回各种“银行”所需要的时间与费用的。例如，将BTC 提出，依然是需要一个小时的时间与几十块的费用的。仔细想想支付宝的大额提现与提现手续费你就懂了吧。

当然，你此时一定会冒出“那，Mixin 不就是一个中心化的节点（平台）了吗？”这样的问题。嗯，你是对的，的确是这样的，只不过 Mixin Network 是一个按去中心化方式运营的中心型节点。至于它本身是怎样去中心化运营的，这个我以后会慢慢讲，有兴趣大家也可以先去读读 Mixin 的白皮书。

此外，很多人有点分不太清楚 Mixin Messenger 与 Mixin Network 的关系。其实 Mixin Network 是一个后端的保存资产与帐户关系的平台，而 Mixin Messenger 则是使用 Mixin Network 接口的官方客户端App。这就如同手机上的支付宝应用 APP 与阿里无数台服务器构成的支付宝平台之间的关系一样。不过比较另类的是 Mixin Messenger 没有作成手机支付宝的样子，而是做成一个聊天工具。哈，这是要支付宝 + 微信合体吗？

Mixin Messenger 伪装成聊天工具的区块链定投神器

发表于 2019-08-30 更新于 2020-12-16

Mixin Messenger 是我一不小心差点错过的一款神器：居然是一个伪装成聊天工具的区块链钱包+交易所+定投工具+理财工具+社区工具的神器！

做为币圈老鸟，早在2017年底我就知道了 Mixin Network 要ICO的消息，不过那时我正在为BTC的价格HIGH的不行，而且除了EOS基本上不参加任何ICO，毕竟有太多的ICO不靠谱，再加上当时20EOS/XIN的价格有点让人感觉太高（哈，我还是太没有想象力了）。

再加上Mixin的技术是使用了另类的 TEE-BFT-DAG 让我一个 BlockChain 的原教旨主义者有点举棋不定，于是也就错过了，甚至在漫漫熊市都没有想起来看它一眼。

哪怕它早在今年年初主网上线了也基本上在业内动静不算太大，毕竟那时正熊得大家灰头土脸的吧。

不过也就是今年的7月3日之后，我终于发现了它的牛逼之处，原来居然是一个最好用与安全的多币种钱包，哈，你居然伪装得这么好，真是服了。

钱包

我在上一篇文章《来，这一次让我们慢慢变富》中说过，长期严肃地数字资产投资最先需要搞定钱包与交易所。但是个人管理钱包其实是有很大的坑的：从比特币创建伊始，私钥的安全保存就是一个极大的难题，有很多人就是因为不小心丢掉了私钥，结果币在那里却取不出来。

其实这主要是因为比特币的设计中有一个不符合常识的地方：

  • 现实世界里每个人是先拥有了一个银行卡号，再设置一个取款密码，然后使用这种方式来管理资产的，如果不慎密码忘了，还可以通过身份证明去银行找回密码。
  • 但是比特币的世界里却不一样：每一个地址（相当于银行卡号）都是根据私钥（相当于取款密码）算出来的，掌握了私钥就掌握了一切。现实世界中你只是丢了取款密码别人不知道你的帐号，还没有办法取走你的钱，但是区块链世界中，如果私钥忘了，是没有一个如同银行一样的中心机构能帮你找回密码的，也就是说没有任何人（对！就是任何人，也包括你自己）能动用那个帐号中的钱；如果是私钥被偷了，也就意味着帐号里的钱也会一起被偷。
  • 更可怕的是：如果你的币被人偷走了，极大可能要不回来，因为比特币的世界里交易是不可逆的，而且地址本身是匿名的。

嗯，我知道你读到这里大概会想问，既然私钥是唯一获取数字资产的凭证，那我怎样保证我的私钥不被人猜中；或者如果我如果猜中了别人的私钥，岂不是直接就能拿走别人的币？
Good Question! 对！说中了要害了！所以为了安全，私钥使用了一个巨大的可能空间，有多大呢？ 2的256次方，相当于10的77次方，也就是1的后面要写77个0才可能表式出来。
嗯，我知道这样说还是不太直观，那再说两个数字吧，地球的原子总数是10的50次方，而可观测宇宙的原子数是10的80次方。
如果地球上的每个原子都是一台计算机，并且每个原子每秒可以算1亿次私钥，那也需要3671亿年才能穷举完所有的私钥，而现在已知的宇宙也只有137亿年而已，那时太阳系是否存在都不一定了。

来，这一次让我们慢慢变富

发表于 2019-08-28 更新于 2020-12-16

至2012年不经意间接触到比特币，从开始的懵懂无知，再至不明觉厉，又至后来的幡然领悟，蓦然回首才发现我已经在比特币的世界里生存了七年了。现在细细想来，不禁一阵阵冷汗，虽然这一路走得跌跌撞撞，错误频频，但好在没有被甩下车，真是庆幸。

在这七年里，除了老婆及几个平时信得过的同事，我倒也真没有向太多人安利过比特币(区块链/数字资产)，主要是这个东西当年来看太颠覆，想说明白需要的认知门槛有点高，我也是看了好多遍的白皮书并参考了一些源代码才能确信自己理解其中的基本原理，而且要向非专业人士解释这件事，还涉及一些软件之外的东西，例如经济学，金融学，博弈论，密码学之类的东西，这些我也不是很懂啊。而且自以为了解是一回事，能向不同知识背景的人说明白又是另一回事，难度不在一个量级上。

不过最重要的原因，比特币也就是“钱”啊。和朋友谈钱或者是投资，哪怕是股票也是一件需要很慎重的事。说者无心，听者也无心，只是吹吹牛，自然也没有啥；不过可怕的是听者有意了，还真的居然拿来做了投资的依据：赚了虽然不会分我，但是亏了却是因为我的推荐，难免被抱怨；更可怕的是如果有一样投资标的你不看好，对方没有买却看着这玩意疯涨了上去，他会认为是你拦着他的财路。唉，我本将心向明月啊、、、

吃过亏之后，我一般遇到这种情况都会闭口不言，或者换个其它的话题，反正吹牛的话题多得是嘛。不过最近两年，也还真有一个小学同学孜孜不倦地问我些比特币投资的事。在确认了对方是认真的之后我给了三个作业：

  1. 搞明白钱包与交易所，试着从交易所买点任何种类的数字资产并提到自己的钱包里
  2. 去“得到”订阅《通往财富自由之路》并在三个月内学习完
  3. 仔细审视自己的金钱观，想明白财富对自己意味着什么，并真的作出为了要实现财富自由会付出多长时间、多少金额的钱来进行定投

不过现在看来这三个作业完成的并不太好，嗯，也在意料之中，毕竟现在的区块链就像2000年的互联网，入门还是不太友好：第一个作业就卡住了半年、、、

虽然我这个作业看来似乎有点不太近乎情理，但其实我是有严肃考量的：

  • 交易所是普通人与区块链世界接触的第一入口，想要严肃地对待区块链投资的话，无论如何也是需要学会亲自保管数字资产的（想想以前各种交易所被盗币的新闻你就会明白了）
  • 投资是需要智慧的，而且几乎所有投资失败的原因都是因为使用了错误的方法，或者是被错误的心理、情绪控制而导致的。尤其是在数字资产这种可7x24交易市场里，如果没有做好足够的心理建设，被割几乎是唯一的结果。
  • 李笑来作为数字资产领域出名的成功投资人，他的经验是不可忽视的，而且他也是国内唯一一个能成体系的将自己的投资经验讲出来，并能教会大家的人
  • 一个老司机的成功投资经验，只要198元的订阅费用其实是超值的；对于不肯付费，也不肯学习的人来说，投资就真的不用做了，唉
  • 虽然所有人都喜欢钱，但是很少人审慎地想过财富对自己意味着什么：是更多的自由？还是达成什么目标的手段？或者只想做一个富有的废人？对于理性的投资还需要想清楚自己能持续多久的投入多少钱，并清楚自己的*风险偏好
  • 没有想清楚上面的问题，哪怕运气好也赚不到大钱；就算赚到了大钱，也守不住啊
  • 定投是唯一直接出坑的方法，只是能做到的人太少

你看，其实如果要严肃地对待投资需要做的事还不少吧。总不能让我直接就回答你：“你就告诉我，比特币现在还能不能买？”这样的问题吧。因为我无论回答你什么问题，最终你都有极大概率做不好，赚不到钱的。

本来我也就想这个事就拉倒吧，教会别人赚钱可能是这个世界上最难的事了，而且我现在也不算有钱人，谁信你个穷货的说教？
